HOW TO RUN THE SMART TRAINING WEBSITE

1. Open XAMPP.
2. Start Apache and MySQL.
3. Copy the smart-training folder into:
   C:/xampp/htdocs/
4. Open phpMyAdmin:
   http://localhost/phpmyadmin
5. Import database.sql.
6. Open the website:
   http://localhost/smart-training/index.php
